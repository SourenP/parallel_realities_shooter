# parallel_realities_shooter

2D Shoot 'Em Up game made with C and SDL2 by following the [parallelrealities.co.uk: SDL2 Game Tutorials](https://www.parallelrealities.co.uk/tutorials/#shooter).

## Run

- Install prereqs
  - `sdl2`
  - `sdl2_mixer`
  - `sdl2_image`

```
make
./shooter
make clean
```
